# @rn-motion-ui/utils

Shared TypeScript utilities for the `rn-motion-ui` monorepo.

## Install

```sh
npm install @rn-motion-ui/utils
# or: bun add @rn-motion-ui/utils
```

## Subpath exports

| Import | Export |
| --- | --- |
| `@rn-motion-ui/utils/typeguards` | Common type-guard helpers (`isString`, `isNumber`, …). |
